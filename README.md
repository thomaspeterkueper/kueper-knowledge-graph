# KUEPER Knowledge Graph

Gemeinsamer Wissenskern für:

- kueper.com — wissenschaftliche Grundlagen und Forschung
- solarsciencefoundation.org — Vermittlung, Lernpfade, Übungen
- overtimearchive.org — epistemisch markiertes Archiv, Real/Fiction-Exploration
- noxiagame — spielerische Anwendung

Der Graph ist nicht Website, Archiv oder Spiel. Er ist die gemeinsame semantische Ebene darunter.

## Grundprinzip

Wissen wird als Entitäten und Beziehungen beschrieben.

Eine Entität kann ein Konzept, Ort, Ereignis, Dokument, Lernmodul, Technologie, Person, Organisation oder Universum sein.

Eine Beziehung verbindet zwei Entitäten und trägt selbst einen epistemischen Status.

## Startstruktur

```text
/standards   kanonische Architektur- und Inventarisierungsstandards
/entities    Entitäten, zunächst Foundation Nodes
/relations   Beziehungen zwischen Entitäten
/documents   Dokument-Knoten und Dokumentregister
/pilots      Arbeitsprotokolle und Pilotinventare
/schemas     maschinenlesbare JSON-Schemas
/imports     spätere Importe aus KUEPER, OTA, SSF, Noxia
/exports     spätere Ausgaben für Websites und Spiel
```
