# KUEPER Knowledge Graph

Gemeinsamer Wissenskern für:

- kueper.com — wissenschaftliche Grundlagen und Forschung
- solarsciencefoundation.org — Vermittlung, Lernpfade, Übungen
- overtimearchive.org — epistemisch markiertes Archiv, Real/Fiction-Exploration
- noxiagame — spielerische Anwendung

Der Graph ist nicht Website, Archiv oder Spiel. Er ist die gemeinsame semantische Ebene darunter.

## Scope

Dieses Repository verwaltet Entitäten, Relationen, Knowledge Domains, KXF-Exports und systemübergreifende Mappings.
